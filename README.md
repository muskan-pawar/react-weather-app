# React Weather App

A simple weather app built with React, Vite and Material UI.

## 🚀 Live Demo
https://react-weather-app-five-theta.vercel.app
## Features
- Search weather by city
- Shows temperature, humidity, min/max temperature
- Weather icons based on conditions
- Clean UI using Material UI

## Tech Stack
- React
- Vite
- Material UI
- OpenWeather API

## Installation

git clone https://github.com/muskan-pawar/react-weather-app.git

cd mini-project-react

npm install

npm run dev
